Scroll Reveal amb animation-timeline: view()

Desplaça't cap avall per veure com els elements apareixen progressivament quan entren al viewport

rocket_launch

Primer Element Animat

Aquesta animació utilitza animation-timeline: view() per activar-se automàticament quan l'element entra al viewport. És una característica moderna de CSS que permet crear efectes de scroll sense JavaScript.

auto_awesome

Segon Element Animat

Cada element apareix amb un efecte diferent: fade-in, desplaçament vertical, escala i rotació. L'animació s'activa progressivament a mesura que fas scroll per la pàgina.

visibility

Tercer Element Animat

L'atribut animation-range permet controlar quan comença i acaba l'animació durant el scroll. En aquest cas, comença quan l'element entra al viewport (entry 0%) i acaba al 30% (cover 30%).

bolt

Quart Element Animat

Aquest efecte combina múltiples transformacions: translateY per al desplaçament vertical, scale per l'escala inicial i opacity per la transparència. Tot això es coordina amb @keyframes.

check_circle

Cinquè Element Animat

Per a navegadors que no suporten animation-timeline: view(), hi ha un fallback implementat amb @supports not que aplica una animació tradicional amb delay progressiu.

info

Fallback implementat amb @supports not (animation-timeline: view()) per garantir compatibilitat amb tots els navegadors.
