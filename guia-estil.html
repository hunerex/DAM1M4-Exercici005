Escala Tipográfica (rem-based)

Extra Small (0.75rem / 12px) - Mínimo recomendado para accesibilidad
Small (0.875rem / 14px) - Textos secundarios y metadatos
Base (1rem / 16px) - Tamaño estándar para cuerpo de texto
Large (1.125rem / 18px) - Texto destacado o introductorio
Extra Large (1.25rem / 20px) - Subtítulos pequeños
2X Large (1.5rem / 24px) - Subtítulos medianos
3X Large (2rem / 32px) - Títulos secundarios

♿ Accesibilidad: El tamaño mínimo recomendado para texto es 12px (0.75rem). El contraste de color también es esencial: un mínimo de 4.5:1 para texto normal y 3:1 para texto grande (18px+).
